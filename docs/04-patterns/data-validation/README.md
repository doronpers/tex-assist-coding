# Data Validation

Learn how to validate user input to prevent errors and security issues.

## Why Validation Matters

**Without validation:**
```python
# User sends garbage → Your app breaks
```

**With validation:**
```python
# User sends garbage → You return helpful error
```

## Key Principles

1. **Never trust user input**
2. **Validate on the server** (not just frontend)
3. **Give helpful error messages**
4. **Use validation libraries** (like Pydantic)

---

**Validate everything that comes from users!** ✅
